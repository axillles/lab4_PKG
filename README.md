# lab4_PKG
# Растровые Алгоритмы

Этот проект демонстрирует реализацию различных алгоритмов растеризации линий и окружностей с использованием библиотеки Pygame. Программа визуализирует работу алгоритмов и измеряет время их выполнения.

## Алгоритмы

1. **DDA (Digital Differential Analyzer)**: Алгоритм для растеризации отрезков.
2. **Алгоритм Брезенхема для линий**: Эффективный алгоритм для рисования линий.
3. **Алгоритм Брезенхема для окружностей**: Используется для рисования окружностей.
4. **Инкрементальный алгоритм**: Простой метод для растеризации линий.

## Особенности

- Визуализация работы каждого алгоритма.
- Осевые линии с размеченными делениями для отображения длины.
- Измерение времени выполнения каждого алгоритма.

## Установка и Запуск

1. Убедитесь, что Python и Pygame установлены на вашем компьютере.

2. Клонируйте репозиторий или скачайте файлы:

    ```bash
    git clone <repository-url>
    ```

3. Перейдите в директорию проекта и запустите скрипт:

    ```bash
    cd <project-directory>
    python main.py
    ```

## Использование

- Запустите программу, чтобы увидеть визуализацию алгоритмов.
- Закройте окно, чтобы завершить выполнение.
